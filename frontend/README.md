# Frontend (React + Vite)

This folder contains the React UI for the Case Management System.

For full project documentation (architecture, backend setup, database seeding, deployment notes), see the root README:

- ../README.md

## Run locally
```bash
npm install
npm run dev
```

By default, the frontend expects the backend API at `http://localhost:5000`.

## Build
```bash
npm run build
npm run preview
```
